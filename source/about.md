Here is a binary search in Python:

```python
def bin_search(arr: list, target) -> int:
    lo, hi = 0, len(arr)
    while lo < hi:
        m = (lo + hi) // 2
        if arr[m] < target:
            lo = m + 1
        else:
            hi = m
    return lo
```
